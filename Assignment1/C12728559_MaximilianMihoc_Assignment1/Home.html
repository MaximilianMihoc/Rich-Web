Quality Assurance - Acuitas and Focus

During his time in Ocuco, Maximilian did a lot of testing. First tests were for a smaller program used in the company and after he gained more experience, more important tests were comming. At a later stage, he started to do regressin testing for the main products.

Regression testing is a type of software testing that seeks to uncover new software bugs, in existing functional or non-functional areas of a system after changes such as enhancements, patches or configuration, have been made. The intent of regression testing is to ensure that changes to the application have not introduced new faults.

Maximilian did regression testing for Acuitas and Focus in both windows 8.1 and windows 10. The bugs found had to be included into a document with clear explanation and sent to the Quality Assurance Team leader. After testing the products separately in different versions of Windoes, a comparison of performance between Windows 7 and Windows 10 has been done.
